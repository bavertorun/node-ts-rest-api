# NodeJS &amp; TypeScript REST API (CRUD)

## Get Started

```
git clone https://github.com/bavertorun/node-ts-rest-api.git
```

```
node-ts-rest-api
```

```
npm install
```

`
open .env file and add your mongodb uri
`

```
npm run start
```
<br><hr><br>

Start Project Dev:

```
npm run dev
```

Start Project:

```
npm run start
```

Build Project:

```
npm run build
```
